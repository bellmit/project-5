1-3a. Check compliance and certification availability (main PS)

Collect all needed documents:

  • MSDS, UN38.3 test report, 1.2M drop test report, air/sea transportation report
  • BQB
  • FCC, NCC
  • FDA, Taiwan FDA
  • CE, RoHS
  • Confirm HS code and check for Import/Export rules (Taiwan; US; UK; CA) i.e. C02, 貨物稅(T / T*)
  • Collect all needed certification for importing (if we dont have the document we cannot return goods to Taiwan)

All compliance and certifications MUST be completed before shipping. They do not have to be completed for approval.
